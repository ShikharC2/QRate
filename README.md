# QRate - Curating the future
## Automatic attendance system

#### QRate solves this gap. We provide every student with a unique QR-based ID, and attendance is marked instantly using a simple web camera scan. 

Attendance is stored securely in the cloud, giving schools reliable digital records and administrators real-time insights. QRate is not just an attendance system — it’s a low-cost digital foundation for smarter, more accountable rural education
